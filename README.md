## Development

### Install
```batch
npm install
```

### Run
```batch
npm run start
```

### Publish to Expo
```batch
exp publish
```

### Build APK
In Terminal 1
```batch
exp start
```

In Terminal 2
```batch
# Starting Build
exp build:android

# Monitoring Status
exp build:status
```

After a while the monitor command will return a URL that you can download the APK from.
